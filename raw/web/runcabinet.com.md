# Cabinet — AI-First Knowledge Base

## Product Overview
**Cabinet** is a free, open-source, self-hosted knowledge base OS designed around AI-first workflows. It combines markdown files on disk with autonomous AI agents, embedded HTML apps, and persistent memory for knowledge work.

## Core Description
Cabinet enables teams and individuals to build an operating system for their work where AI agents have a persistent brain—a knowledge base that remembers context across sessions. As described: "An AI-first knowledge base where files live on disk and a team of AI agents helps you execute."

## Key Features

- **File-Based Storage**: Markdown files on disk with no database, ensuring portability and git-backed version control
- **AI Agent Orchestration**: Create custom AI agents (CEO, Editor, Marketer, etc.) with goals, skills, and scheduled jobs
- **Embedded HTML Apps**: Drop `index.html` into folders to render interactive dashboards and tools
- **Web Terminal**: Full xterm.js terminal access within the browser interface
- **WYSIWYG Editor**: Rich text editing with markdown toggle capability
- **Scheduled Jobs**: Cron-based automation for agents to work 24/7
- **Git Integration**: Auto-commits on every save; linked repo support via `.repo.yaml`
- **Full-Text Search**: Instant fuzzy search across all pages
- **PDF & CSV Support**: Native inline rendering and editable tables
- **Team Chat**: Internal channels with agent @mentions
- **Mission & Task System**: Kanban boards and progress tracking

## Real-World Use Cases

- B2C app management
